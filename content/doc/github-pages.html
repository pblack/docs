In the _config.yml file, add the variable baseurl (without trailing slash):


baseurl: "/project-name"

In your templates, use the site.baseurl variable for assets:


<link rel="stylesheet" href="{{ site.baseurl }}/css/styles.css">

When forestry generates your previews we will set the baseurl to empty.  You can also do this when you run Jekyll, locally:


$ jekyll serve --watch --baseurl=
