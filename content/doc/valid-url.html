When setting up a site in Forestry, you'll need to ensure that there's a URL set in your config file.

Jekyll

Hugo

With Jekyll, you'll need to set the url variable in your config file. 

url: http://mysite.com
or

url: http://localhost:4000		For Hugo, you'll need to set the baseurl variable in your config file. If you're using toml, it should look like this:

baseurl: "http://mysite.com"
or

baseurl: "http://localhost:1313" 
For more clarification on the difference between  url and baseurl in the Jekyll config, see Parker Moore's post.
